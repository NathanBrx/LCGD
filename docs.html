Introduction

Welcome to the documentation of the LCGD Domain Specific Language !

Players

Players are the main actors of the game.

  • Each player has a unique identifier.
  • Players have a color property, and can be given custom attributes like hp or powers.
  • Players can interact with the game by moving pawns.

Pawns

Pawns are the base element of any game.

  • Each pawn has a unique identifier.
  • Pawns have properties such as category, position, sprite, and player.
  • Players interact with the pawns by moving them.
